# SUD data for version 2.9

All UD (version 2.9), except the 5 ones coming from native SUD (`UD_Beja-NSC`, `UD_French-GSD`, `UD_French-Rhapsodie`, `UD_French-ParisStories` and `UD_Naija-NSC`) are converted in SUD format with the [UD &rarr; SUD conversion system (tag 2.9)](https://github.com/surfacesyntacticud/tools/tree/v2.9/converter) (usable with the [Grew](https://grew.fr) software).

Hence, the data below are built with:
  * The 5 native SUD described above
  * 212 corpora converted:
    * from [version 2.9 of UD corpora](http://hdl.handle.net/11234/1-4611)
    * with [tag `v2.9` of the conversion system](https://github.com/surfacesyntacticud/tools/tree/v2.9/converter)

## Download all corpora
Download the full set of 217 SUD corpora: [sud-treebanks-v2.9.tgz](https://grew.fr/download/sud-treebanks-v2.9.tgz).
